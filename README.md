# Basic Concepts

## Shift

A shift is a set of works that are each assigned any number of members.

## Member

A member can be anything as long as it has the ability to participate in works.

## Work

A work also can be anything from little chore to large project.

## Group

A group is an arbitrary set of members.

## Vacation

A vacation is a period for which the member is not able to participate in works.

# Options

## Member

### Initial Load

It represents the load or hardship imposed upon the member BEFORE the current shift starts. The member with high initial load will be given less works in terms of their relative difficulties.

## Vacation

### Paid

The vacation is said to be a paid one if the member is not going to be imposed more works later due to their vacancy.
